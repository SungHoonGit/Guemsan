# Guemsan

<h3> CRM이란? </h3>
CRM은 고객 관계 관리를 의미합니다. CRM 시스템을 사용하면 고객 데이터와 고객 대면 액티비티(판매, 마케팅, 서비스, 전자 상거래)를 자동화하고 통합할 수 있습니다.
